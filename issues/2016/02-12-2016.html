Version Updates

0.38


Highlights

  • Show Warning for Background Remote Debuggers
  • Adding Jest preset so that people can configure Jest using react-native as preset
  • Use yarn when available
  • Fallback to JS Animation if native animated is not present
  • Upgrade to lodash@^4.16.6, results in faster packager and smaller npm install size
  • Documentation updates, including better user experience, more detailed explanations and more
  • Integration test stability: add delay before starting TimersTest
  • Modernize AppContainer and add rootTag in the child context

0.39


Highlights

  • TouchableOpacity: Respond instantly to first touch
  • Add the configuration option to the run-android command
  • Add extractOffset to Animated
  • Add a new Navigator SceneConfig swipeFromLeft
  • Add aspectRatio style property
  • Only skip updating computed flex basis within the same generation
  • Rewrite 'react-native init' and 'react-native upgrade' without using Yeoman in preparation for templates support
  • Make Modal pass rootTag to AppContainer
