React Native June 2017 (v0.46.0) released

BREAKING CHANGES

  • Better TextInput: contentSize property was removed from TextInput.onChange event.
  • Using Image with nested content now causes yellowbox warning.

BUGFIXES

  • Fix stickySectionHeader re-render issue. 
  • Fix KeyboardAvoidingView component.
  • Replace React.PropTypes with prop-types.
  • Replace React.createClass with create-react-class.
